# BrightTV-Viewership-Analytics-
This project analyzes BrightTV user engagement and viewership data to support subscription growth and customer value management (CVM). The goal is to understand viewing behavior, identify key content drivers, and uncover opportunities to improve engagement, retention, and audience expansion.

# Problem Statement
BrightTV aims to grow its subscription base, but user engagement is uneven across content, time periods, regions and demographics. A root cause analysis is required to understand what drives engagement and where growth opportunities exist.

# Aim
The aim of this project is to provide data driven insights to support subscription growth and improve customer engagement strategies.

# Objectives
Identify key content and engagement drivers
Understand user viewing behaviour and patterns
Analyze demographic and geographic distribution
Identify high-value customer segments
Provide actionable recommendations for growth

# Methodology
1. Project Planning
Structured analysis approach using a logical CVM framework
Defined key business questions around engagement, content, and retention

2. Data Processing (SQL- Databricks)
Cleaned and transformed raw user and viewership data
Handled missing values and standardized fields
Aggregated data by:
Content channels
Time of day
User demographics
Geographic regions

3. Data Analysis
Performed exploratory data analysis
Identified patterns in:
Viewing behaviour
Peak engagement times
High-performing content
Segmented users into value-based groups

4. Visualization
Built charts to present insights clearly
Developed a presentation for analysis, findings and recommendations.

# Key Insights
SuperSport Live Events, Trace TV, and SuperSport Blitz are the top content drivers
Afternoon is the peak viewing period across the platform
Core audience consists of working age users (30–44 years)
Gauteng dominates viewership, followed by Western Cape
Male users show higher engagement overall
Black viewers represent the largest audience group
A High Customer Value Segment drives high engagement

# Project Structure
Project Overview & Dataset – Case study background
Data Processing (SQL-Databricks) – Cleaning and transformation using databricks
Project Planning-Miro Mindmaps( Process and data)
Data Analysis – Excel pivots tables and charts
Presentation – Insights and recommendations

# Conclusion
BrightTV engagement is driven by live sports content, daytime viewing patterns and a concentrated working age audience. While the platform shows strong core performance, there are clear opportunities to expand into new demographics and content categories. Leveraging high value users and improving personalisation can significantly enhance subscription growth and long term user retention.
